# Textavia MCP

MCP server exposing Textavia tools generated from the shared registry.

```sh
npx @textavia/mcp
```

MCP Registry name:

```text
io.github.caravaca-labs/textavia-mcp
```

## Privacy

Default tools run locally. Network tools are hidden unless explicitly enabled by
the host configuration.
