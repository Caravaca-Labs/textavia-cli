# Textavia Privacy Scrubber Skill

Use this skill to redact likely-sensitive text locally.

## CLI

### Privacy scrub

```bash
txv text privacy-scrub "email a@b.com from 1.2.3.4"
```

## Online tool

[https://textavia.com/tools/data-privacy](https://textavia.com/tools/data-privacy)

## Notes

- Runs locally by default.
- Use `--json` for structured agent output.
- Use canonical tool IDs with `txv run <tool-id>` for automation.
