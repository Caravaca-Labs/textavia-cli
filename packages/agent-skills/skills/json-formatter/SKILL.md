# Textavia JSON Formatter Skill

Use this skill to format, minify, and validate JSON locally.

## CLI

### Format JSON

```bash
txv json format '{"a":1}'
```

## Online tool

[https://textavia.com/tools/json-formatter](https://textavia.com/tools/json-formatter)

### Minify JSON

```bash
txv json minify '{"a": 1}'
```

## Online tool

[https://textavia.com/tools/json-minify](https://textavia.com/tools/json-minify)

### Validate JSON

```bash
txv json validate '{"a":1}'
```

## Online tool

[https://textavia.com/tools/json-validator](https://textavia.com/tools/json-validator)

## Notes

- Runs locally by default.
- Use `--json` for structured agent output.
- Use canonical tool IDs with `txv run <tool-id>` for automation.
